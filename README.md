# Exploratory Data Analysis (EDA) Project

## Objective
Analyze Titanic dataset to identify patterns and trends.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Findings
- Survival depended on passenger class.
- Female passengers survived more.
- Age influenced survival.

## Visualizations
- Survival Count
- Gender Distribution
- Age Histogram
- Correlation Heatmap
